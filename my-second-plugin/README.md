# my-second-plugin

A plugin built with the PlugCLI framework.

## Setup
Everything is ready! This project is self-contained.

## Build your plugin
```powershell
.\build-plugin.ps1
```

## Run
```powershell
java -jar plugcli-core-1.0.0.jar sample YourName
```

## Available commands
```powershell
java -jar plugcli-core-1.0.0.jar help
```
